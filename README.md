# AppDescktop-V2
Es la continuación de las practicas para aprender a desarrollar aplicaciones de escritorio en C# con la librería .NET.

## Resumen
### Estructura de datos
- #### Array
    Los arrays o "arreglos" son una colección de elementos, en el caso de c# deben ser **ELEMENTOS DEL MIMSMO TIPO**.
    Para acceder a cada elemento tenemos que indicar el número de la posición del elemento dentro del array, a esto se le denomina **ÍNDICE**.<br>
    Los índices o posiciones se inician contando desde el índice 0, que sería el primer elemento en adelante, así como se muestra en el siguiente ejemplo
    
    |     0       |    1       |     2      |    3     |   4      |...|
    |:-----------:|:----------:|:----------:|:--------:|:--------:|:-:|
    |Elemento 1   |Elemento 2  |Elemento 3  |Elemento 4|Elemento 4|...|

- #### Matrices
    Al igual que el array, las matrices son una colección de elementos, pero lo que lo distingue, es que las matrices se componen de filas y columnas.<br>
    Es similar en cuanto a una tabla, para acceder a alguno de sus elementos, se debe de indicar la posición en la fila y columna.<br>
    Una representación gráfica de las matrices sería la siguiente:

    |Filas\Columnas |0                  |1                  |2                  |3                  |...|
    |:-------------:|:-----------------:|:-----------------:|:-----------------:|:-----------------:|:-:|
    |0              |Elemento 1 (fila 0)|Elemento 2 (fila 0)|Elemento 3 (fila 0)|Elemento 4 (fila 0)|...|
    |1              |Elemento 1 (fila 1)|Elemento 2 (fila 1)|Elemento 3 (fila 1)|Elemento 4 (fila 1)|...|
    |2              |Elemento 1 (fila 2)|Elemento 2 (fila 2)|Elemento 3 (fila 2)|Elemento 4 (fila 2)|...|
    |3              |Elemento 1 (fila 3)|Elemento 2 (fila 3)|Elemento 3 (fila 3)|Elemento 4 (fila 3)|...|
    |..             |...                |...                |...                |...                |...|
